# TaskFlow DB 🚀

TaskFlow DB is a lightweight, responsive task management solution featuring cloud-powered data persistence.

## Features
*   **Vanilla JS Frontend**: Clean, modern, responsive UI with light/dark theme support.
*   **Node.js & Express REST API**: Production-grade backend server handling CRUD, search, and paginated task requests.
*   **MongoDB Atlas Persistence**: Cloud storage utilizing Mongoose ODM, validation constraints, and optimized compound indexes.
*   **Offline Fallback Mode**: Transparently switches to browser `localStorage` if the server is offline or unreachable, ensuring continuous operation.
